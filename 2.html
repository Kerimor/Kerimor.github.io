Разработка программ:

Разработку ПО начнем с графического интерфейса. Это связано с особен-ностями программирования для Android. Для того, чтобы взаимодействовать с элементами окна, программный модуль подключается к необходимым элемен-там с помощью идентификаторов. При этом элементы интерфейса в главном окне должны быть созданы в первую очередь. Как уже отмечалось ранее, гра-фический интерфейс пользователя объявляется в XML файлах, а основным языком для разработки приложений Android является объектно-ориентированный язык программирования Java. В соответствии с определенной структурой при разработке компонентов приложения будет использован язык Java. В качестве среды разработки используется среда AndroidStudio, так как для нее предоставляется плагин для работы с Android SDK, а также она является свободно распространяемой.
