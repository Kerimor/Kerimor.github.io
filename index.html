Разработка игрового приложения под операционную систему Android

Цель работы – создание игрового приложения для операционной системы Android.

Для достижения этой цели необходимо решить следующие задачи:

№1

Изучить область игровых приложений для мобильных устройств.

№2

Рассмотреть вопрос о разработке программ для операционной системы Android и использовать их для реализации приложения.

№3

Рассчитать экономические показатели программного продукта.

№4

Выполнить описание техники безопасности программиста при разработке приложения.

ЗАДАЧИ

Область игровых приложений

Игровые приложения распространенны и актривно используются как на плотформе Android, так и на iOS, Windows, Linux и т.д.

Разработка программ

Мобильные приложения можно разработать при помощи множества языков программирования на множестве приложений.

Экономические показатели

Мобильный гейминг является самым прибыльным среди всех платформ, рассмотрим денежные показатели в зависимости от операционной системы.

Техника безопасности

При разработке мобильного приложения необходимо соблюдать правила, стобы обеспечить безопасность как для разработчика, так и техники.
